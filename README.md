📱 Project: Ksheera-Sagara – Dairy Profit/Loss Calculator

💡 Problem:
Many dairy farmers track milk production but don’t know their actual profit after expenses — leading to unnoticed losses.

🔍 Solution I Built:
An Android app that helps farmers:
✔️ Track daily milk data (Liters & Fat%)
✔️ Log expenses (fodder, medical, labor, electricity)
✔️ View clear Profit/Loss dashboard
✔️ Analyze performance per cow

⚙️ Development Highlights:
• Built using Kotlin (Android Development)
• Used Room Database for data storage
• Leveraged Antigravity AI for faster development & problem-solving 🤖
• Tested on a real device using WiFi Debugging 📲

🌱 Impact:
Transforming dairy farming into a data-driven and profitable business

This project enhanced my skills in mobile app development, debugging, and applying AI-assisted development tools.

Grateful for the opportunity and learning experience 🙌
